Fast Accesible Setup Tools

How To Use

  1. Create a new folder: Choose or create an empty folder where you want your project to live.
  2. Open PowerShell in that folder: Right‑click inside the folder and select Open in Terminal or PowerShell.
  3. Run the setup script: Paste the following commands into your PowerShell window, this will download all required files and dependencies.
  4. Initialize the project: After the script finishes, double‑click on SetupProject.bat in your folder. This will configure the project structure and get everything ready to build.

# sdl-vsbt-setup

A minimal setup for building SDL3 projects using Visual Studio Build Tools from the command line.

curl.exe -L -o "$(Get-Location)\codeDL.ps1" "https://raw.githubusercontent.com/Kiliken/sdl-vsbt-setup/main/codeDL.ps1"
powershell.exe -NoProfile -ExecutionPolicy Bypass -File "$(Get-Location)\codeDL.ps1"
